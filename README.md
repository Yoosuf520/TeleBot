# TeleBot: AI-Powered Telegram Assistant 🤖

A modern, asynchronous Telegram chatbot built with **Python**, **aiogram 3.x**, and the **OpenAI API**. This project demonstrates the integration of Large Language Models (LLMs) into a real-time messaging interface with a focus on clean architecture and security.

## 🚀 Features
- **Asynchronous Architecture:** Built using `aiogram 3.x` for high-performance handling of concurrent users and non-blocking I/O.
- **Intelligent Conversations:** Powered by OpenAI's GPT models to provide natural, context-aware responses.
- **Secure Configuration:** Implements `python-dotenv` for environment variable management, ensuring API keys remain private.
- **Modern Python Standards:** Utilizes type hinting and the latest `aiogram` Dispatcher/Router patterns.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Framework:** [aiogram 3.x](https://docs.aiogram.dev/)
- **AI Engine:** OpenAI API (GPT-3.5/GPT-4)
- **Environment:** Conda / Virtualenv
